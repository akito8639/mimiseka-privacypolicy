プライバシーポリシー

最終更新日: 2026年3月28日

ミミセカ（MimiSeka）は、写真や映像をもとにAIが情景を読み取り、文学的な物語や要約テキストを生成する iOS アプリです。生成されたテキストは、写真のキャプション（ALT テキスト）として埋め込んだり、音声で朗読したりすることができます。

  • ミミセカ シア（Scia）: 撮影した写真をもとに、情景の要約や物語を生成します。生成されたテキストは ALT テキストとして写真に保存したり、音声で読み上げることができます。
  • ミミセカ ルペ（Lupe）: 視覚に制約のある方を主な対象とした、リアルタイムカメラ読み取り機能です。カメラを向けた先の文字や物体を AI が即座に認識し、音声で説明します。

本ポリシーでは、これら2つのアプリにおけるデータの取り扱いについて説明します。

1. 収集するデータと利用目的

写真データ

写真のキャプションとなる要約や物語を生成するために、撮影または選択された写真を外部AIサービスに送信します。送信後のデータの取り扱いは、各AIサービスのプライバシーポリシーに従います。

  • ミミセカ シア: 生成された要約や物語をメタデータとして写真データに埋め込み、端末の写真ライブラリ（「ミミセカ」アルバム）に保存します。この機能は設定で無効にできます。
  • ミミセカ ルペ: 撮影された写真は、端末の写真ライブラリには保存しません。

初回アンケート

アプリの初回起動時に、以下の情報をお尋ねします。

  • 視覚の状態
  • 年齢
  • 物語モードの好み
  • 自動読み上げの設定

これらの情報はアプリの設定を最適化するために使用されるほか、サービス向上を目的として匿名化のうえ本アプリのサーバーに保存されます。取得した情報が特定の個人を識別するために使用されることはありません。

デバイス識別子

サービスの安定運用（1日あたりのリクエスト数制限）のために、端末固有の識別子（identifierForVendor）をサーバーに送信します。この識別子はアプリを再インストールすると変更され、他のアプリや広告目的では使用されません。

位置情報（任意）

位置情報の利用は初期設定で無効になっており、設定画面から有効にした場合にのみ取得されます。取得された位置情報は以下の目的で使用されます。

  • 写真に位置情報メタデータを付与する（Scia でアルバム保存時）
  • 物語生成の文脈情報として利用する

位置情報はアプリのメモリ上に一時的にキャッシュされるのみで、サーバーには送信されません。

利用状況データ

アプリの改善のために、以下の利用状況データを端末内に記録します。

  • 物語生成の回数や完了率
  • 各機能の使用頻度
  • エラーの発生状況
  • アクセシビリティ機能の利用状況（VoiceOver 等）

これらのデータは匿名化のうえ本アプリのサーバーに保存され、サービス向上の目的にのみ使用されます。特定の個人を識別するために使用されることはありません。

データの保持期間

利用状況データおよびデバイス識別子は、サービス提供期間中保持され、サービス終了時に削除されます。初回アンケートデータは匿名化された統計情報として、サービス提供期間中保持されます。

2. 外部サービスとの連携

AI サービス

写真の解析および物語の生成のために、以下のAIサービスを利用しています。すべての通信は本アプリのサーバー（AWS Lambda）を経由し、暗号化された接続（HTTPS）で行われます。

  • OpenAI — 写真の解析、テキスト生成
  • Anthropic（Claude） — テキスト生成

送信後のデータの取り扱いは、各サービスのプライバシーポリシーに従います。

  • OpenAI プライバシーポリシー
  • Anthropic プライバシーポリシー

音声合成サービス

生成されたテキストを音声に変換するために、以下のサービスを利用する場合があります。送信されるのはアプリ内で生成されたテキストのみで、写真やユーザーの識別情報は含まれません。

  • ElevenLabs — 要約や物語テキストの発話用音声合成
  • Aivis Cloud — 要約や物語テキストの発話用音声合成
  • ElevenLabs プライバシーポリシー
  • Aivis Cloud プライバシーポリシー

X（旧 Twitter）

X への投稿機能は任意です。利用する場合、X の OAuth 認証を通じてアカウントに接続します。本アプリが使用する権限は投稿（ポスト）のみであり、タイムラインやフォロワー情報などの読み取り権限は取得しません。認証トークンは端末内のキーチェーンに安全に保存され、ログアウト時に削除されます。

3. 写真の保存とメタデータ

シア（Scia）

生成された要約・物語テキストは、写真のメタデータ（IPTC キャプション）として埋め込み、端末の写真ライブラリ内の「ミミセカ」アルバムに保存できます。この機能は設定で有効・無効を切り替えられます。保存される写真には以下のメタデータが含まれます。

  • 要約・物語テキスト（ALT テキストとして）
  • 感情タグ（選択した場合のみ）
  • 視覚レベル・物語モード
  • 位置情報（位置情報機能を有効にしている場合）

ルペ（Lupe）

カメラで捉えた映像や文字を AI が即座に読み取り、音声で読み上げます。解析に使用した画像は端末の写真ライブラリには保存されません。

4. データの安全性

  • すべての外部通信は HTTPS で暗号化されます
  • API キーはアプリ内に保持せず、サーバー側で管理しています
  • Apple App Attest による端末認証を実施し、不正なアクセスを防止しています
  • 認証トークンやデバイス識別子は、端末のキーチェーンに安全に保存されます

5. 第三者への提供

上記の外部サービスを除き、本アプリが収集したデータを第三者に販売、貸与、または提供することはありません。

6. 国際データ移転

お客様のデータは、外部のAIサービスおよび音声合成サービスにより、お客様の居住国以外の国で処理される場合があります。すべてのデータ転送は、当社サーバーを経由し、暗号化された接続（HTTPS）で行われます。

7. お子様のプライバシー

本アプリは特定の年齢層を対象としたものではありませんが、視覚に制約のあるお子様にもご利用いただけます。13歳未満であることが示された場合、アンケート回答やデバイス識別子など、個人を特定しうる情報をサーバーに収集・送信することはありません。サービスの運用・改善に必要な匿名の利用状況データ（個人を特定できない統計情報）は収集する場合があります。アプリの主要機能（写真解析、物語生成、音声読み上げ）は引き続きご利用いただけます。

万が一、13歳未満のお子様の個人情報を意図せず収集していたことが判明した場合、速やかに削除いたします。お心当たりのある場合は、privacy@mimiseka.jp までご連絡ください。

8. ユーザーの権利

  • アプリ内の設定画面から、位置情報の利用やアルバム保存などの機能をいつでも無効にできます
  • X 連携はログアウトすることで認証情報を削除できます
  • アプリを削除することで、端末内のすべてのデータが削除されます（写真ライブラリに保存された写真を除く）

9. ポリシーの変更

本ポリシーを変更する場合は、アプリ内またはウェブサイトでお知らせします。重要な変更がある場合は、アプリ起動時に通知します。
